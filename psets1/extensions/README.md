# [File Extensions](https://cs50.harvard.edu/python/2022/psets/1/extensions/)
a program that prompts the user for the name of a file 
and then outputs that file’s media type if the file’s name ends, 
case-insensitively, in any of these suffixes:

## Supports
.gif
.jpg
.jpeg
.png
.pdf
.txt
.zip

## Getting Started
### Prerequisites
- [Python](https://www.python.org/).

### Installation
```
git clone https://github.com/Ranjit-Pokharel/CS50-Python-Problem-Sets.git
cd CS50-Python-Problem-Sets/psets1/extensions
python extensions.py
```
